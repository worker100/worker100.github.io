## 🔧 配置说明

### API配置
后端服务默认运行在 `http://localhost:3001`，提供以下接口：
- `GET /api/ai-news` - 获取AI资讯列表
- `GET /api/ai-news/hot` - 获取热门资讯
- `GET /api/ai-news/stats` - 获取统计数据
- `POST /api/ai-news/refresh` - 手动触发更新

### 自定义设置
在 `js/ai-news-updater.js` 中可配置：
- 更新频率（默认24小时）
- 缓存数量（默认100条）
- 显示数量（默认20条）
